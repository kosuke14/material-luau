## Components

| Component | Description |
|---|---|
| Badge | Notification indicator dot or count |
| Card | Content container (Filled, Outlined, Elevated) |
| Checkbox | Toggle selection with checkmark |
| Chip | Compact element for filters, actions, or selections |
| CircularProgress | Indeterminate circular loading indicator |
| Dialog | Modal prompt with headline and buttons |
| Divider | Visual separator between content |
| FloatingActionButton | Prominent action button |
| Icon | Material Symbols icon |
| IconButton | Icon-only action button |
| Item | List item with icon and label |
| LinearProgress | Horizontal progress bar |
| Menu | Dropdown menu attached to a button |
| NavigationBar | Bottom navigation bar |
| NavigationTab | Tab for NavigationBar |
| RadioButton | Single selection from a group |
| Ripple | Material ripple touch feedback |
| Shadow | Elevation shadow effect |
| SideSheet | Side panel overlay |
| Snackbar | Temporary bottom message with action |
| Switch | Toggle selection on/off |
| TabItem | Individual tab content |
| Tabs | Tab container |
| TextButton | Button with text label |
| TextField | Text input (Filled, Outlined) |

## Tokens

MaterialLuau ensures that it is up to Material Design 3 specs by using actual Token values.

## Theming

MaterialLuau allows you to fully customize your Material Design 3 colors and typography using actual JS code from Google.
